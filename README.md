# Zepto_SQL_Analysis
Data Analytics Project – SQL-Based Business Insights using PostgreSQL
1. Overview
This project demonstrates an end-to-end data analytics workflow using PostgreSQL.
The objective was to import a structured dataset into PostgreSQL, perform Exploratory Data Analysis (EDA), clean and transform the data, and solve real-world business problems using SQL queries.
The final output includes actionable insights derived from query results to support data-driven decision-making.

This project highlights strong fundamentals in:
Database management
SQL querying
Data cleaning and transformation
Business-oriented analytical thinking

2. Dataset
Source: Structured dataset (CSV format)
Data Type: Transactional / Business data
Size: Medium-scale dataset suitable for SQL analysis

The dataset required preprocessing due to:
Missing values
Duplicate records
Inconsistent formatting
Incorrect data types

3. Tools & Technologies
PostgreSQL – Database management and SQL querying
pgAdmin – Query execution and database management
SQL – Data cleaning, transformation, aggregation, and analysis
Excel / CSV – Initial dataset format

4. Project Steps
Step 1: Database Setup
Created a new database in PostgreSQL
Defined appropriate table schema
Imported dataset using COPY command

Step 2: Data Cleaning
Checked for NULL values
Removed duplicate rows
Standardized column formats

Step 3: Exploratory Data Analysis (EDA)
Performed SQL queries to:
Count total records
Identify distinct categories
Calculate summary statistics (SUM, AVG, COUNT, MIN, MAX)
Examine distribution of key metrics

Step 4: Business Problem Solving
Used SQL queries to answer business-focused questions 


5. Results & Key Insights

The SQL analysis revealed:

High-revenue products driving majority sales contribution
Revenue concentration within specific categories
Identification of underperforming segments
Actionable insights for improving profitability and optimization

These insights can support:
Strategic pricing decisions
Inventory planning
Marketing focus areas
Customer targeting strategies
